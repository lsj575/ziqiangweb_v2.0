我们的团队

一代代前行，沐浴自强社光

办公室

我们是自强君的小秘书，我们是将情报的收集、消息的传达、活动奖品的购买、同时与各个部门的联系和系列社团活动及例会内容的记录集一身的神秘小组织。我们在社团活动中起着非常重要的作用哦。

宣传部

顾名思义，负责社团内的宣传活动。主要是负责海报宣传单和视频的制作，以及官Q微信公众号的推送.（也就是说自强君就是我们）另外，部内也有500G的百度云资源，当然了，是宣传资料~

策划部

策划部就是一个为各种活动提供精确计划的部门，需要有对活动经费的预算能力以及有应对各种突发事件的对策.总的来说，需要一个清晰的头脑和灵活的应变能力。小伙伴们有没有兴趣来考验一下自己咩~

外联部

我们外联部就是跟外界联络的一个部门.平时的主要任务就是借教室、联络老师啥的，加上蜜汁拉赞助.希望各种帅锅美铝正太萌妹属性的学弟学妹们，能来我们外联部，我们在这儿等着你 ~

义工服务队

义工的小队，技术的殿堂! ! !这里有学长带你们体验acm的乐趣，有义务维修电脑的成就感，有网页制作的团结互助，当然自强社的活动也少不了我们的参与，我们是一个积极向上的小团体。
